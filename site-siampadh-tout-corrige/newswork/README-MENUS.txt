NOUVEAU SYSTEME DE MENUS

Les 7 jours de la semaine sont affichés côte à côte dans le site.
Il n'est plus nécessaire de descendre dans le PDF pour voir toute la semaine.

Pour modifier les menus visibles dans le calendrier :
1. Ouvrir le fichier menu-week-data.js
2. Modifier les plats dans la partie correspondant au type de menu
3. Enregistrer le fichier

Les PDF complets restent dans : assets/menus-pdf
