COMMENT MODIFIER LE SITE DEPUIS GITHUB

1) Modifier l'équipe
- Ouvrir le fichier equipe-data.js sur GitHub
- Cliquer sur le crayon ✏️ Edit
- Modifier les noms, fonctions et textes
- Pour une photo : envoyer la photo dans assets/team puis écrire son chemin dans photo
  Exemple : photo: 'assets/team/sophie.jpg'
- Cliquer sur Commit changes

2) Modifier les actualités
- Ouvrir le fichier actualites-data.js sur GitHub
- Cliquer sur le crayon ✏️ Edit
- Modifier ou ajouter une actualité
- Pour une photo : envoyer la photo dans assets/actualites puis écrire son chemin
  Exemple : photo: 'assets/actualites/sortie-ete.jpg'
- Cliquer sur Commit changes

3) Modifier les menus
- Ouvrir menu-week-data.js
- Modifier les plats de la semaine
- Cliquer sur Commit changes

Après chaque modification, attendre 30 secondes à 2 minutes puis actualiser le site.
