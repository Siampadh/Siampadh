COMMENT UTILISER LA PARTIE ÉQUIPE

- L’équipe est visible par tout le monde sur le site.
- Le bouton « Modifier les photos / les mots » ouvre seulement la partie modification.
- Le mot de passe sert uniquement à modifier, pas à voir l’équipe.
- Vous pouvez ajouter/changer les 7 photos de profil : 4 intervenantes, Sophie Leudière, Eddy Lila et Pierre Bacqué.
- Après enregistrement, les modifications restent mémorisées dans le navigateur utilisé.

Mot de passe administrateur : gobelets

Important : sur un site 100 % HTML/CSS/JS, les modifications sont sauvegardées dans le navigateur avec localStorage. Pour que les changements soient visibles sur tous les ordinateurs, il faudra ensuite brancher un vrai hébergement avec base de données ou un espace admin serveur.
